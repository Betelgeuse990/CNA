# Métodos Numéricos

Repositório para as atividades da disciplina de Cálculo Numérico Aplicado (CNA).

## Sobre
Este repositório contém implementações desenvolvidas para a disciplina de CNA em que cada pasta corresponde a uma tarefa (tipicamente os PPCs) que incluem:

- O código-fonte em Python;
- Um arquivo README explicando o problema e a abordagem de solução;
- Gráficos de saída, quando aplicável.

## Versão Python e biblioteca(s) usada(s)
- Python 3.10+
- Matplotlib
- Math (no momento, apenas para constantes matemáticas)
